Features
--------

A list of some interesting features:

* OAuth authentification.
* UTF-8 support.
* Tweets, Retweets, Follow, Unfollow.
* Mentions, Reply
* Search
* Direct messages
* Transarency and colors.
* Based on ncurses.
* Customisation of keys binding.
* Customisation of colors.
* Easely configurable, readable configuration.
* Opens URL just with single key.

[IMPORTANT]
This project is relatively new, started in May 2011, and curently in heavy
development, there is lots of planned functionality still to be coded.
