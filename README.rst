Description
-----------

Tyrs is a twitter client based on curse easely configurable.

Functionnality
--------------

- **Oauth authentification**
- utf-8 support
- Display home timeline with colorfull tweets
- Send tweets
- Retweets (TODO)

For more information, check doc/ files.
